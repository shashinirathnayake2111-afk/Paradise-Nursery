# 🌿 Paradise Nursery - Tropical Houseplant Store

Paradise Nursery is a premium, modern e-commerce web application dedicated to providing high-quality tropical houseplants. The app features a stunning landing page with natural animations and an intuitive shopping experience localized for the Sri Lankan market.

## ✨ Features

- **🌸 Interactive Landing Page**: Features a "Natural Entrance" animation where leaves drift and sway into place.
- **🖼️ Premium Product Gallery**: A curated collection of plants (Air Purifying, Aromatic, Tropical) with high-quality images and elegant hover effects.
- **🛒 Real-time Cart Management**:
  - Add/Remove items with a single click.
  - Live quantity updates with subtotal calculations.
  - Fully localized currency support in **Sri Lankan Rupees (Rs.)**.
- **📱 Fully Responsive**: Optimized for all devices, from mobile phones to high-resolution desktops.
- **✨ Modern Aesthetics**: Implemented with glassmorphism, editorial typography (Lora & Inter), and organic transitions.

## 🛠️ Technology Stack

- **Frontend**: React.js
- **State Management**: Redux Toolkit
- **Styling**: Vanilla CSS (Modern CSS3)
- **Icons**: Lucide React
- **Build Tool**: Vite

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/paradise-nursery.git
   cd paradise-nursery
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open in browser**:
   Navigate to `http://localhost:5173` (or the port specified in your terminal).

## 📁 Project Structure

- `src/assets/`: Contains images and static assets.
- `src/Components/`:
  - `ProductList.jsx`: The main shop gallery.
  - `CartItem.jsx`: The interactive shopping cart.
  - `AboutUs.jsx`: Information about the nursery.
- `src/store/`: Redux store and Cart slices for state management.
- `src/plants.js`: Centralized data file for all plant products.
